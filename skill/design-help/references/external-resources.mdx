## Typography

### Font Pairing

- [Fontpair](https://fontpair.co/) — Curated Google Font pairs

### Scale & Rhythm

- [Type Scale](https://type-scale.com/) — Visual type scale generator

### Readability

- Readable body text: 16–20px on desktop
- Line length: 65–75 characters
- Line height: 1.5–1.75 for body text

## Color

### Palette Generation

- [Coolors](https://coolors.co/) — Palette generator
- [Huemint](https://huemint.com/) — AI-assisted brand colors
- [Realtime Colors](https://realtimecolors.com/) — Preview colors in context

### Accessibility

- [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/) — WCAG compliance
- [Stark](https://www.getstark.co/) — Figma/Sketch accessibility plugin
- [Color Safe](http://colorsafe.co/) — Accessible color combinations

### Color Theory

- Build tints (white) and shades (black) from base
- Consider color blindness (~8% of men)
- Use semantic naming (success, warning) not color names (green, yellow)

## Accessibility

### Standards

- [WCAG 2.1](https://www.w3.org/WAI/WCAG21/quickref/) — Web accessibility guidelines
- [WAI-ARIA](https://www.w3.org/WAI/standards-guidelines/aria/) — Accessible rich applications

### Quick Checks

- [ ] All text meets 4.5:1 contrast ratio
- [ ] Interactive elements have focus states
- [ ] Forms have labels (not just placeholders)
- [ ] Images have alt text
- [ ] Color isn't the only indicator
- [ ] Keyboard navigation works

- [Tailwind Documentation](https://tailwindcss.com/)

### Design Tokens to Tailwind

```js
// tailwind.config.js
module.exports = {
  theme: {
    colors: {
      primary: { /* palette */ },
      neutral: { /* palette */ },
    },
    fontFamily: {
      sans: ['Inter', 'system-ui', 'sans-serif'],
    },
    spacing: {
      // 4px base scale
    }
  }
}
```
